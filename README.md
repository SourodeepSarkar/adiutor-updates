# Adiutor v1
![Platform](https://img.shields.io/badge/platform-macOS%2014%2B-black?style=for-the-badge&logo=apple) ![SwiftUI](https://img.shields.io/badge/Built%20With-SwiftUI-orange?style=for-the-badge&logo=swift)

First public release of Adiutor — a native macOS study environment focused on intelligent focus tracking, analytics, and distraction-aware productivity workflows.

Built entirely with SwiftUI and designed around a local-first architecture.

---

## Highlights

### Intelligent Study Tracking
- Tracks usage of selected study applications
- Real-time monitored sessions
- Focus-oriented analytics engine
- Local-only tracking architecture

### Native macOS Experience
- Fully native SwiftUI interface
- Apple-inspired glassmorphism UI
- Smooth animations and adaptive layouts
- Optimized for modern macOS workflows

### Analytics Dashboard
- Daily study duration insights
- App usage distribution charts
- Focus statistics visualization
- Historical session analytics

### Statistics Export System
- Automatic JSON snapshot generation
- Local statistics storage
- External analytics dashboard support
- CSV and PDF export workflows

### Setup Experience
- Guided onboarding flow
- Study app selection
- Personalized workspace initialization
- Accessibility permission integration

### Built-In Help System
- Dedicated Help window
- Keyboard shortcut support
- Workflow guidance

### Sparkle Integration
- In-app update system via Sparkle
- GitHub-based update distribution
- Version-aware update delivery

- SwiftUI

---

## Privacy

Adiutor is fully local-first.

No telemetry.
No cloud sync.
No external analytics collection.

All study data remains on-device.

---

## Requirements

- macOS 14+
- Accessibility permissions required for tracking functionality

---

## Known Limitations

Because Adiutor is currently distributed without Apple Developer notarization:

- macOS Gatekeeper warnings may appear
- First launch may require:
  Right Click → Open

This does not affect functionality.

---

## Installation

1. Download `Adiutor.zip`
2. Extract the archive
3. Move Adiutor.app into Applications
4. Launch Adiutor
5. Grant Accessibility permissions when prompted
